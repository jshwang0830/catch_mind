# 포트번호 입력하는 초기 창
 - 포트번호를 입력하면 회원 및 입장 유저들과 게임 진행에 대한 관리가 가능한 화면으로 이동
 - 포트번호 입력 필드 및 생성,나가기 버튼 

# 관리 모드
 - 유저 목록 확인 가능
 - 현재 입장 유저 확인 가능
 - 유저들의 모든 활동(채팅,게임)을 실시간으로 받아 확인 가능
 - 위 기능들을 수행하기에 최소 3개의 텍스트필트가 필요할 듯.
 - 나가기 버튼을 기본으로 나머지 버튼은 생각해볼 예정

<Client 용>
# 초기 게임 입장 창
 - 포트 번호, IP, 아이디, 패스워드 입력란 
 - 회원가입 버튼, ,연결 설정 버튼, 로그인 버튼, 나가기 버튼
 - 처음에 IP와 포트 번호만 입력받고, 연결설정이 완료되면 로그인을 위한 아이디 패스워드 텍스트 필드가 활성화 되고 로그인 버튼으로 바뀜
 
# 회원 가입 창
 - 회원가입 버튼 클릭 시 입장
 - 기존 창은 유지하면서 회원 가입창을 띄움
 - 회원 가입에 필요한 필드 작성 후 나가기 및 가입 버튼 생성
 - 가입 버튼 클릭 시 입력된 정보들로 텍스트 파일 생성 및 데이터 저장
 
# 대기실 창
 - 유저목록, 채팅창, 게임룸 // 또 있나..?
 - 유저목록의 유저에게 쪽지 보내기 기능
 - 채팅창을 이용한 대화 가능.
 - 방을 개설해서 게임룸으로 입장.
 - 리스트에서 게임룸에 입장하여 게임창으로 이동
 
# 게임창(미구상)
 - 6인~8인까지 입장 가능하게 구상 중
 - 채팅창을 이용할지, 말풍선을 이용한 채팅을 이용할지 생각 중
 - 센터에 그림 그리는 필드를 만듦 그 외 캐치마인드 게임 진행 구성대로 진행
 - 아직 미구상
